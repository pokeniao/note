-  字段规范

| 功能    | 字段         |
| ----- | ---------- |
| 插入，添加 | addXXX     |
| 获取    | getXXX     |
| 修改    | alterXXX   |
| 方法    | XXXMethod  |
| 布尔变量  | XXXBoolean |
| 显示开关  | XXXVisible |

# 后端规范
- 按照`get` `post` `put` `delete` 顺序
- ==插入==的Service业务返回插入行`ID`


# 数据库设计规范
- 字段用烧烤串 \_分割
